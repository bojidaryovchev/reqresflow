# Flow Tabs

Flow tabs work similarly to request tabs but are for editing and running flows.

## Open a Flow Tab

1. Click a flow name in the sidebar, or create a new flow.
2. If the flow is already open in a tab, the app just switches to that tab.
3. If not, a new tab opens showing the flow editor.

## Switch Between Flow Tabs

1. Click on any flow tab at the top of the screen to view it.

## Close a Flow Tab

1. Click the **×** on the flow tab, or middle-click the tab.
2. The tab closes. If it was the active tab, the nearest remaining tab becomes active.
3. If no flow tabs remain, the area shows as empty.

## Close All Flow Tabs

1. Right-click on any flow tab.
2. Click **Close All Tabs**.
3. All flow tabs are closed.

## Duplicate a Flow from a Tab

1. Right-click on a flow tab.
2. Click **Duplicate Flow**.
3. A copy of the flow is created (with "(copy)" in the name).
4. The copy opens in a new tab.

## Right-Click Menu on Flow Tabs

1. Right-click any flow tab.
2. Options: **Duplicate Flow**, **Close Tab**, **Close All Tabs**.
3. Click anywhere else to dismiss the menu.

## How the Tab Bar Switches Between Requests and Flows

1. When you're in the **Flows** sidebar section:
   - The tab bar shows only your flow tabs.
   - The **+** button creates a new flow.
2. When you're in **Collections** or **History**:
   - The tab bar shows only your request tabs.
   - The **+** button creates a new request.
3. Both sets of tabs exist at the same time — switching sidebar sections just changes which ones you see.
