# Keyboard Shortcuts

## Save (`Ctrl+S`)

1. Press `Ctrl+S` on your keyboard (or `Cmd+S` on Mac).
2. If your request is already saved to a collection:
   - Your changes are saved immediately.
3. If your request hasn't been saved yet:
   - A popup appears asking you to choose which collection to save it to.

## Send Request (`Enter`)

1. Click on the URL input field.
2. Press `Enter`.
3. The request is sent — same as clicking the "Send" button.

## Renaming (`Enter` / `Escape`)

1. When you're editing a name (collection, request, flow, or payload):
   - Press `Enter` to save the new name.
   - Press `Escape` to cancel and keep the old name.
