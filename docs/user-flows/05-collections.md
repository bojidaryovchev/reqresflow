# Collections

Collections are folders where you save and organize your requests.

## Create a Collection

1. In the left sidebar, make sure you're on the "Collections" section.
2. Click the **+** button next to the "Collections" header.
3. A new collection appears called "New Collection".
4. The name is immediately editable — type a name and press Enter.
5. The collection is saved automatically.

## Rename a Collection

1. Find the collection in the sidebar.
2. Click the pencil icon (**✎**) on the collection.
3. Type the new name.
4. Press Enter to confirm, or Escape to cancel.

## Delete a Collection

1. Click the **×** icon on the collection you want to delete.
2. The collection and all its requests are removed immediately.

## Save a Request to a Collection

**If your request is already saved to a collection** (you'll see a "Save" button):

1. Make your changes to the request.
2. Click **Save** (or press `Ctrl+S`).
3. Your changes overwrite the previously saved version.
4. The unsaved dot (**●**) disappears from the tab.

**If your request is new** (not yet saved anywhere):

1. Click **Save** (or press `Ctrl+S`).
2. A "Save to Collection" popup appears.
3. Choose which collection to save it to by clicking on one.
4. The request is added to that collection.
5. From now on, pressing Save will update it in place.

**Alternatively, from the sidebar:**

1. Click the **+** icon on any collection in the sidebar.
2. The current request is saved to that collection as a new entry.

## Open a Saved Request

1. In the sidebar, expand a collection by clicking its name.
2. Click on a request to open it.
3. If you already have that request open in a tab, the app just switches to that tab.
4. If not, a new tab opens with all the request's settings loaded.

## Delete a Request from a Collection

1. Hover over the request in the sidebar.
2. Click the **×** icon that appears.
3. The request is removed from the collection.

## Rename a Request in the Sidebar

1. Click the pencil icon (**✎**) on the request.
2. Type the new name and press Enter.
3. If the request is open in a tab, the tab name updates too.

## Rename a Request from the Tab Area

1. With a request tab active, look for the "Request name..." field at the top.
2. Type a new name there.
3. If the request is saved to a collection, the name updates there too.

## Expand and Collapse Collections

1. Click a collection name to expand it and see its requests.
2. Click again to collapse it.
3. The collection containing your currently active request auto-expands when you open the app.
